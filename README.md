# What is this?

Get perfect every time for the non-designer.

# Installation

`npm i shadow --save`

Then...

...
import {shadow} from 'shadow';

shadow({
    shadow_type: 'soft';
    padding: false
});

...

##  Options

Shadow supports  a few options to customize how it works.

* *shadow_type* - _hard | soft_ (Defaults  to "soft")
   
* *padding* - _boolean_  (Defaults to `false`)
